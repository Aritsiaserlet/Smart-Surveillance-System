ระบบ Dashboard มอนิเตอร์กล้องวงจรปิดอัจฉริยะ ระดับองค์กร (Enterprise Grade) ที่ผสานพลัง AI เพื่อการตรวจจับเหตุการณ์ผิดปกติแบบ Real-time พร้อมระบบจัดการสิทธิ์ผู้ใช้งานและแผนที่อาคารแบบ Interactive

#Key Features

Smart Camera Grid: สลับเลย์เอาต์การมองเห็นได้ทั้ง 3x3 และ 5x5 พร้อมสถานะกล้อง (Live, Alert, Offline, AI Processing)
Interactive Mini-Map: แผนที่อาคาร 2D/3D ที่แสดงตำแหน่งกล้องและจุดเกิดเหตุ (Incident) พร้อม Preview ภาพสดเมื่อเอาเมาส์ชี้
AI Alert Chat (High Severity): ระบบแชทแจ้งเตือนเหตุการณ์ผิดปกติ แบ่งระดับความรุนแรง (Info, Warning, Critical, Emergency) พร้อม Snapshot และค่าความมั่นใจ (Confidence Score)
Analytics & Insights: กราฟสรุปจำนวนคนในอาคาร (Occupancy), ช่วงเวลาที่มีคนเยอะ (Peak Time) และคำแนะนำจาก AI (Actionable Recommendations)
Granular Permissions: ระบบจัดการสิทธิ์ผู้ใช้งานรายบุคคลผ่าน Gmail สามารถเปิด-ปิดสิทธิ์การเข้าถึงแยกตามฟังก์ชัน (Live, Playback, Export)
Advanced Video Player: ระบบเล่นวิดีโอย้อนหลัง (History), สลับโหมด Real-time, ระบบ Snapshot และ AI Overlay (Bounding Boxes, Heatmap, Object Tracking)
